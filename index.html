Hướng dẫn kết nối Google Sheets

Quan trọng: Nếu đã deploy trước đây, bạn cần copy lại code mới bên dưới và deploy lại (New deployment) để tính năng Nhật Ký Hành Động hoạt động. Sheet "Nhật Ký Hành Động" sẽ được tự động tạo khi có thao tác đầu tiên.

Bước 1: Tạo một file Google Sheets mới (Đặt tên: Pivo S ATELIER).

Bước 2: Vào menu Tiện ích mở rộng (Extensions) ➔ Chọn Apps Script.

Bước 3: Xóa sạch code cũ, copy đoạn mã bên dưới và dán vào. ⚠ Phải dùng code mới nhất này.

Bước 4: Nhấn Triển khai (Deploy) ➔ Chọn Triển khai mới (New deployment).

Bước 5: Chọn loại Ứng dụng khách (Web app). Thiết lập quyền truy cập "Ai cũng có quyền" (Anyone) ➔ Nhấn Triển khai.

Bước 6: Copy đường link Web app URL mới dán vào ô cấu hình bên trái và bật Auto-Sync.

✅ Sau khi deploy xong, nhấn nút "Test ghi nhật ký hành động" để kiểm tra.
